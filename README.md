# mender_x86_conversion

The goal of this project is to document how to convert a stable linux x86 image to a mender image. Final deliverable is step-by-step instructions on how to set up the initial image and the build steps for further os-level and application-level updates.

## Milestones

Milestones:
1. set up initial image
1. convert image

### Initial Image

Create an image from a stable debian-based distribution. It needs to be partitioned so that mender-convert can work with it.
